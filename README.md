# Lista de Exercícios — Programação Estruturada em C/C++

Este projeto contém todos os exercícios práticos e avaliações realizados ao longo da disciplina, organizados por tema e complexidade crescente.  
Os exercícios cobrem desde lógica básica até vetores, matrizes, funções, estruturas heterogêneas (structs) e ponteiros.

---

# Observação
Todos os arquivos estão com extensão `.cpp` porque foram desenvolvidos no Visual Studio, mas utilizam conceitos de **Programação Estruturada** equivalentes aos da linguagem C.
